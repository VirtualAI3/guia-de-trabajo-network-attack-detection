## Para descargar el repositorio hagan lo siguiente (SOLO LA PRIMERA VEZ)
```bash
git clone https://github.com/VirtualAI3/guia-de-trabajo-network-attack-detection.git
```
## PARA SUBIR EL DOCUMENTO
```bash
git pull
git add .
git commit -m "[Nombre] modifico el documento"
git push
```
* **Nota**: si no funciona git push, usar git push origin main
